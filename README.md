## 🚀 Quick Start

```bash
npm install
npm run dev          # localhost:3000
npm run build        # production build → out/
```

## 🔄 Deploy

```bash
# Preview (GitHub Pages)
git push origin staging

# Production (emxsports.com)
npm run deploy:prod
```
